Direction Field Plotter

An Interactive Applet powered by Sage and MathJax.

(By Kelsey Norman with HTML code from Prof. Gregory V. Bard)

\textrm{Given a } y', \textrm{ a function of } x \textrm{ and } y, \textrm{ this demo plots the direction field generated by } y'. \\ \textrm{Then, given a set of initial conditions where } init y = f( init x ), \textrm{ the demo plots the curve through that point that fits the direction field. }
